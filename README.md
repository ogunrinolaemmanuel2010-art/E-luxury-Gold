# E-gold Luxury

A single-page luxury gold marketplace demo with authentication, product browsing, cart, checkout, order history, theme switching, and receipt email support.

## Project Structure

- `index.html` - page markup and SVG icons
- `styles.css` - visual styling and responsive layout
- `script.js` - app state, cart, checkout, orders, and UI behavior

## Run Locally

Open `index.html` in a browser, or serve the folder with a local static server.
